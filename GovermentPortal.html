Accessibility Features

This portal is built to meet WCAG 2.1 AA standards. Below are key accessibility features implemented:

1. Correct Color Contrast Ratio

All text meets minimum contrast ratios for readability.

2. Descriptive Links

Link text clearly describes the destination or action.

3. Keyboard Compatible

All functionality available via keyboard navigation.

4. Understandable Images

All images have appropriate alt text or descriptions.

5. Clear Navigation

Consistent, predictable navigation structure.

6. Correct Header Tags

Proper heading hierarchy for screen reader users.

7. Animation and Scrolling

Motion controls and no auto-playing content.

8. Personalized Text

Text can be resized without loss of functionality.

9. Well Thought Page Titles

Descriptive page titles that identify the page purpose.
